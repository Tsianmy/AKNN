# Project of AKNN

This is a project of AKNN.

### Requirements and Dependencies

- **Requirements:**
	- A C++ compiler

### DataSet

**SIFT1M** and **GIST1M** from http://corpus-texmex.irisa.fr/.

You need to put the dataset to the directory AKNN/sift and AKNN/gist.

Additionally, **KNN-graphs** built on the dataset are needed. Their paths should be AKNN/sift/sift_100NN_100.graph and AKNN/gist/gist_100NN_100.graph.
